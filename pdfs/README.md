# Folder PDFs

Folder ini berisi file-file PDF yang akan di-hosting melalui GitHub Pages.

## Struktur File

Simpan semua file PDF Anda di folder ini. Setiap file akan dapat diakses melalui URL:

```
https://rrd27.github.io/Infografis-Digital-Desa-Jajarwayang/pdfs/nama-file.pdf
```

## Penamaan File

Gunakan konvensi penamaan yang baik:
- Gunakan huruf kecil
- Ganti spasi dengan tanda hubung (-)
- Hindari karakter khusus
- Gunakan nama yang deskriptif

### Contoh Penamaan yang Baik:
- `peta-desa-jajarwayang.pdf`
- `laporan-keuangan-2024.pdf`
- `struktur-organisasi.pdf`
- `data-demografi-2024.pdf`

### Contoh Penamaan yang Harus Dihindari:
- `Laporan Keuangan.pdf` (ada spasi)
- `file#1.pdf` (karakter khusus)
- `doc.pdf` (tidak deskriptif)

## Cara Menambahkan File

1. Upload file PDF ke folder ini
2. Edit `index.html` di root repository untuk menambahkan link ke file baru
3. Commit dan push perubahan
4. Tunggu beberapa menit untuk GitHub Pages mengupdate

## Ukuran File

GitHub memiliki batasan ukuran file:
- Maksimal 100 MB per file
- Repository total sebaiknya tidak melebihi 1 GB

Jika file terlalu besar, pertimbangkan untuk mengompresnya terlebih dahulu menggunakan tools seperti Ghostscript atau Adobe Acrobat.
