# Placeholder untuk File PDF

Folder `pdfs/` ini siap untuk menampung file-file PDF Anda.

## Contoh File yang Bisa Ditambahkan:

1. **peta-desa.pdf** - Peta wilayah desa
2. **data-demografi.pdf** - Statistik penduduk
3. **struktur-pemerintahan.pdf** - Bagan organisasi
4. **program-kerja.pdf** - Program dan kegiatan

## Cara Upload:

### Melalui Git:
```bash
git add pdfs/nama-file.pdf
git commit -m "Menambahkan file PDF baru"
git push
```

### Melalui GitHub Web Interface:
1. Buka repository di GitHub
2. Navigasi ke folder `pdfs/`
3. Klik "Add file" > "Upload files"
4. Drag & drop file PDF Anda
5. Commit changes

Setelah file di-upload, Anda dapat mengaksesnya di:
